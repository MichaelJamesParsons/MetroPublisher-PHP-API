# setPathHistory( array $pathHistories )

Sets the path history for this tag.

Sets the path history for this tag.

This method will replace all path_history entries for this content item with the list of path history items. To remove all path history items, call this method with an empty array as the first parameter.

To add many entries at once, fetch this content's path history entries, then merge your own entries with the results and pass the collection into this method.
