To obtain more detailed information about the object, a separate API call must be made. These additional fields are called "meta fields".

Returns the object's meta fields.

Returns the object's meta fields.

Meta fields provide additional information about an object, such as SEO meta descriptions, HTML content of an article/book/review or other content object, and other fields.

These fields will not be returned when fetching groups of objects. A separate request must be made to retrieve these fields.
